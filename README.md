# AI Görsel Prompt Rehberi

Bu repo, GitHub Pages üzerinde yayınlanan tek sayfalık bir HTML rehber içerir.

## Dosyalar

- `index.html`: Yayınlanan ana sayfa
- `CNAME`: GitHub Pages özel alan adı ayarı (`nefisebora.digital`)
- `.nojekyll`: Statik dosyaların doğrudan servis edilmesi için

## Yayın

Site GitHub Pages üzerinden `main` branch kök dizininden yayınlanır.

Canlı adres:

- `https://nefisebora.digital`

## Not

Özel alan adının çalışması için DNS tarafında GitHub Pages kayıtlarının doğru tanımlı olması gerekir.
